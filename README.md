# Train CIFAR10 with quantized ResNet
This Repository was forked from kuangliu!

## Prerequisites
- Python 3.6+
- PyTorch 1.0+

## Training
```
# Start training with: 
python quant_train.py

# You can manually resume the training with: 
python quant_train.py --resume --lr=0.01
```
